# Niche Content Engine - Monetization Roadmap

This document outlines the strategic steps to transition from code generation to real, automated profit.

## Phase 1: Engine & Content (Current)
*   [x] **Scraper Integration**: Real-time discovery of high-growth keywords from Google Trends and TikTok.
*   [x] **Autonomous Reasoning**: AI agents (Groq/Gemini) generating 1,000+ word SEO articles.
*   [x] **Social Pulse**: Automated posting to Instagram and TikTok to drive instant traffic.
*   [x] **History Sync**: Persistent storage to ensure the engine builds onto a growing library of content.

## Phase 2: Deployment & Indexing
*   [ ] **Server-Side Deployment**: Move from `localhost` to Vercel/DigitalOcean.
*   [x] **Fast-Track Indexing**: Google Indexing API notifying search bots of every new article created.
*   [ ] **Custom Domain**: Connect a professional `.com` to build brand authority.

## Phase 3: Immediate Passive Income (Affiliates)
*   [ ] **Amazon Associates**: Continue inserting your `AFFILIATE_TAG` into every article.
*   [ ] **Link Cloaking**: Refine affiliate links to be more "shareable" for higher CTR.
*   [ ] **Click Tracking**: Monitor which niches are actually converting into Amazon sales.

## Phase 4: Long-Term Passive Income (AdSense)
*   [ ] **Domain Approval**: Submit the live site to Google AdSense once it has 20+ quality articles.
*   [ ] **Ad Integration**: Inject the AdSense snippet into `app/layout.tsx`.
*   [ ] **Real Analytics**: Connect the dashboard to the real AdSense Management API to see daily earnings.

## Phase 5: Scale
*   [ ] **Bulk Operations**: Scale from 1 article per hour to 10 articles per hour.
*   [ ] **Niche Expansion**: Automated discovery of "Low Competition / High RPM" niches.
